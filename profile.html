Emotion Recognition

Facial Emotion Recognition using Deep Learning

An interesting field of research with applications in human-machine interactions, safety, and health is facial expression-based automatic emotion recognition. The goal of this project is to create methods for deciphering and coding facial expressions in order to extract information that will help computer systems make more accurate predictions. By utilizing deep learning's numerous architectures and success, our goal was to improve emotion recognition performance. Face detection, feature extraction, and emotion categorization are steps in the process. With the help of the FER2013 dataset, we constructed a Convolutional Neural Network (CNN) architecture and trained the model, obtaining an astounding accuracy of 95.83%. We also included the ability to use a webcam to immediately take pictures and forecast emotions, with graphical results displayed.

ShareRide

Share Ride Application using Django

In order to facilitate users sharing car trips with other people going the same way, the Share Ride Application (SRA) was created. Users have two options: they can ride with another eager user or share their car. The Google Maps API and the Django framework were used in the application's design and development. For office commuters who want to split the cost of their journey and use shared routes, this is very advantageous. Users can choose to give a ride, which lowers their costs.In order to improve the rider experience, we included a suggestion system, rating system, and feedback mechanism in this project. The purpose of these additions was to boost the application's user base and enhance user satisfaction.

Memotion

Memes Emotion analysis using Deep Learning

The Sentiment Analysis for Memes project examined and categorized the emotions conveyed by the image captions of memes. Two models were used to do this: one for text classification and the other for captioning images. I wrote code to caption images using the Inception V3 model for feature extraction and a feed-forward network combined with a recurrent neural network (RNN) to predict the next word in the sequence.The project's goal was to ascertain if a sender's sharing a meme out of happiness, anger, or neutrality. This part of sentiment analysis, sometimes referred to as opinion mining, is identifying and classifying opinions as neutral, negative, or positive within a given text.Emotion analysis has grown in importance as a tool for businesses and its impact on society due to the advancements in technology.
